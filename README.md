# React Frontend Application

This is the frontend UI built with React and Vite.

## Requirements

- Node.js 18+
- npm

## Setup

Clone the repository:

git clone https://github.com/joelgaikwad-125/capstone-frontend.git

cd capstone-frontend

Install dependencies:

npm install

Run the app:

npm run dev

Application runs at:

http://localhost:5173
